# Install Pycharm IDE + Robot Plugin
- Go to [jetbrains.com](https://www.jetbrains.com/)
- Click Tools > Pycharm
- Download & install community edition (free)
- Install *Pycharm*
- Launch Pycharm
- Open File > Settings dialog
- Click on Plugins
- Search for & install *Intellibot* (patched for SeleniumLibrary)
- Restart Pycharm
